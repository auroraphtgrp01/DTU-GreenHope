# 📄 Dịch vụ LCDP

## 📝 Tổng quan
Dịch vụ LCDP (Nền tảng Phát triển Low-Code) được xây dựng sử dụng NocoBase để hỗ trợ phát triển ứng dụng nhanh và quản lý cấu hình cho nền tảng cứu trợ thiên tai Hi Vọng Xanh.
